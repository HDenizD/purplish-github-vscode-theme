# Purplish — GitHub VS Code theme

A more purple take on GitHub's VS Code theme.

This is a small customization of the excellent
[**GitHub VS Code theme**](https://github.com/primer/github-vscode-theme) by
GitHub / Primer. **All the design and the heavy lifting belongs to the original
theme** — this fork only swaps out a few colors so it leans more purple.

## Themes

This fork ships three themes — nothing else:

- **Purplish GitHub Light** — light background with purple/coral accents
- **Purplish GitHub Dark** — neutral near-black background (`#0d0d0d`)
- **Purplish GitHub Dark Purple** — dark aubergine background (`#100d17`)

## What's different

Compared to the original GitHub theme, only a few colors are changed —
everything else (syntax highlighting, terminal colors, diff colors) stays
exactly like the original. All three themes share the same accents:

- 🟣 **Primary buttons are purple instead of green** — the green action buttons
  (Comment, Merge, Checkout, etc.) now use `#8345ff`.
- 🟣 **Purple UI accents** — focus ring (including the Command Palette), badges,
  progress bar, links, list highlights, and the text cursor & selection are
  purple instead of blue.
- 🪸 **Coral for uncommitted files** — untracked/added files in the Explorer and
  Source Control are `#f78166` instead of green.

The two dark variants differ **only** in their background:

| Variant | Editor | Sidebar |
| --- | --- | --- |
| Dark | `#0d0d0d` | `#080808` |
| Dark Purple | `#100d17` | `#0a0710` |

The light theme keeps its light backgrounds and just gets the purple/coral
accents on top.

## Build it yourself

The theme files in `themes/` are generated from the sources in `src/`:

1. Install the dependencies with `npm install` (or `yarn`).
2. Run `npm run build` to regenerate the themes.
3. Run `npm run package` to build an installable `.vsix` into `build/`.
4. Press `F5` in VS Code to open a window with the extension loaded, then pick
   one of the `Purplish GitHub …` themes.

The color customizations live in [`src/colors.js`](src/colors.js) (the two dark
variants share `applyDarkPurplish()`). The icon is generated by
[`scripts/generate-icon.py`](scripts/generate-icon.py) and the preview images by
[`scripts/generate-previews.py`](scripts/generate-previews.py) (`python3` +
`Pillow`).
